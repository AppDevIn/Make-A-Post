## Installation 

### Download 

```bash
git clone https://github.com/AppDevIn/Make-A-Post.git
```

### Change directory 

```bash
cd Make-A-Post
```

### Upgarde flutter

```bash
flutter upgrade 
```

### Get all package

```bash
flutter pub get
```

### See the devices you want to run in

```
flutter devices 
```

### Run the project

```
flutter run -d {device id}
```
